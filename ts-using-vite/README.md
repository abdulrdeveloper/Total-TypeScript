# TypeScript with Vite (Frontend)

Vite is a blazing-fast build tool for frontend web development. This setup is perfect for building React, Vue, or Vanilla TS apps for the browser.

## How to Set This Up Yourself
1. Scaffold a project: 
npm create vite@latest my-app 
- 
-- template: vanilla-ts
2. Navigate to it: cd my-app
3. Install packages: 
npm install

## How to Run
- **Dev Server**: 
npm run dev (Starts a local server with Hot Module Replacement).
- **Build**: 
npm run build (Compiles TS down to optimized static JS/HTML/CSS).
- **Preview**: 
npm run preview (Test the built production files locally).
